Thomas Louvigné  publié le 2020-01-24 inclus dans  catégorie Technique
  • TOUT peut se faire dans le terminal ⤵
  • Il y a un terminal dans EMACS ⤴
  • EMACS est légé 🌬 En terme de RAM & co, j'ai pas vu mieux
  • EMACs est un packet-manager (melpa) ultra riche (4415 packages)
  • EMACS est un logiciel ET un language de programmation, ca peut parraitre mindblow, mais c'est la puissance de LISP .
  • EMACS est ultra moderne: Tout ce que font les IDE de mes collègues, je peux le faire
  • EMACS est ultra vieux 👴. Né en 1976, il n'a pas arreté d'evolué et de s'enrichir
  • EMACS possède un psychologue à qui vous pouvez parler (M-x doctor)
  • EMACS intègre une todo-list ☑ ( org-mode ) simple et génial
  • EMACS me permet de générer ce blog, grace à org-mode ⤴
  • EMACS permet de faire des powerPoint présentations, grâce à org-mode ⤴ et à revealJs
  • EMACS gère les emoji 🦋 (M-x butterfly)
