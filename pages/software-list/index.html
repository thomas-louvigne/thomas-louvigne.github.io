Software List

Voici la liste des logiciels que j'utilise.

Dernière mise à jour : `08/09/2023`

Dot-stuff

Vous pouvez trouver mes fichiers de configuration ici : https://github.com/thomas-louvigne/dot-stuff

PC (Laptop et Fixe / Perso)

Distribution

  • Manjaro
  • Windows Manager : KDE
  • Gestion du son Pipewire
  • Gestion de l'affichage Wayland sur le laptop / X11 sur le PC fixe

Terminal

  • zsh : Terminal
  • oh my zsh : Pour la configuration de zsh ➡ plugins:(git emacs ssh-agent zoxide extract common-aliases git-extras)
  • Kitty: Emulateur de terminal qui fait le taff
  • the_silver_searcher : grep en mieux
  • tree : Jolie affichage de l'arborecence dans linux
  • exa : Remplacement de ls en rust
  • zoxide : Permet de faire du jump dans les dossiers. Je l'utilise en remplacement de cd
  • tldr : Permet d'obternir un man simplifié pour les commandes qu'on utilise
  • xsel : Permet de faire des copier / coller depuis le terminal

Code

EMACS

  • emacs-purcell : Starter pack qui installe pleins de plugin tout en respectant la philosophie et les bindings vanilla
  • Flycheck-grammalecte : Correcteur de grammaire francais
  • Le reste de ma configuration est disponible ici

Gaming

  • Steam : Plateforme d'achat de jeux dématérialisés
  • ProtonQT : Pour gérer les versions de Proton-GE (qui permet des versions améliorées / plus récentes de Proton)

Navigation Web

  • Firefox
  • Brave : En alternative, notamment pour utiliser `teams` au boulot.

Extension pour Firefox

  • u block origin : Anti-publicité
  • Sponsor Block : Permet de bypasser les moments sponsorisés dans les vidéos youtube. Vraiment indispensable.
  • Language tools : Correcteur orthographe / grammaire ( ca marche pas bien avec les google slide)
  • KDE Connect : Controler son PC avec son téléphonne
  • Multi-Account Container : Permet de gérer plusieurs identités sur son navigateur (perso, taff…)
  • Bitwarden : Gestion des mots de passe
  • Bypass PayWalls : Bypasser de paywall de journeaux
  • I still don't care about cookies : Permet de fermer les popups de cookies en les refusants
  • Minimal : Permet d'avoir un internet dépolué des couleurs vives, des popups a la con et des trucs qui prennent votre attention
  • EFF Privacy Badger : Permet de blocker des trackers. C'est géré par l'EFF , donc j'ai plutot confiance
  • Don't fuck with past : Permet de bypasser les site qui bloque le copier / coller

Media

  • playerctl : Permet de commander sa musique depuis le terminal
  • mplayer / mpv et ses gui : Permet de regarder des vidéos, bien mieux que vlc
  • spotify : Musique sur internet
  • GoXLR-utility : Pour gérer ma carte son dans linux
  • The Gimp : Pour modifier des images. Ca reste ok pour bon nombre de trucs basiques

Bureautique

  • Suite google (Google Agenda, Google drive, Google docs, etc…)
  • Libre Office : Bureautique (Moins utilisé)

Cloud

  • NextCloud (Uniquement pour le partage de fichiers)
  • Google Drive (Partage de fichiers)
  • Google Photo (Stoquage et Partage de photos)

Création vidéo

  • obs-studio : Logiciel de streaming. J'utilise la version AUR de tytan652 qui intègre de nombreux plugins.
  • KDEnlive : Montage vidéo
  • Bitwig : DAW / Mixage et enregistrement de musique (style abelton) qui marche très bien avec les périfériques MIDI et avec pipewire.

Thème

Pour KDE, emacs et Kitty, j'utilise le même thème de couleur Dracula.

Sur le site du theme, on peut trouver bon nombre de dot-file pour toutes ces applications.

Autres

  • Bitwarden : Gestionnaire de mots de passe

  • Google Play Store : Pour toutes les application prorpriétaires / obligatoire (WhatsApp & co)

Déplacement

  • Google Maps : En dehors de Paris
  • Trainline : Achat de billet de train

Bureautique

Multiple Factor Authentification (MFA)

  • Application native des GAFAM : Twitter / Facebook / Instagram / Reddit

Messageries

  • WhatsApp, telegram, signal, Facebook Messenger

Media

Web

Autre
